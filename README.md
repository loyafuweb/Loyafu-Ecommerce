# Loyafu E-E-commerce Project

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features
- **Modern UI**: Tailored with Tailwind CSS for a vibrant, mobile-first experience.
- **Product Management**: Browse catalog, filter by category/price, view details.
- **Shopping Cart**: Manage items, adjust quantities, toggle currency (USD/VES).
- **Whatspp Checkout**: Seamless manual order placement via WhatsApp integration.

## Project Structure
- `src/app`: App Router pages.
- `src/components`: Reusable UI components.
- `src/store`: Zustand state management.
- `src/lib`: Utilities.

## Configuration
- **Tailwind**: Custom theme configuration in `tailwind.config.ts`.
- **Images**: Remote patterns configured for Unsplash in `next.config.ts`.
